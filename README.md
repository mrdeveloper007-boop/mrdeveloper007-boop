### 📦 Featured Shopify Apps (Private — built for client stores)

> **CartSense** — turn silent abandonment into actionable insight
> Lightweight cart-feedback prompt + structured drop-off reasons (price/shipping/payment) in an embedded Admin dashboard.
> *Stack:* Remix · Polaris · App Bridge · Admin GraphQL API · theme app extension

> **Low Stock Inventory Tracker** — never let a bestseller go out of stock
> Background watcher on inventory webhooks; pings the merchant when a SKU dips below a configurable threshold.
> *Stack:* Node.js · Remix · Admin GraphQL API · `inventory_levels/update` webhooks · embedded Polaris UI

> **CronPilot** — scheduled store automations without leaving Shopify
> Native cron-job runner for inventory syncs, scheduled publish/unpublish, metafield refreshes, draft order creation.
> *Stack:* Node.js · Remix · Polaris · Admin GraphQL API · server-side cron worker · App Bridge
